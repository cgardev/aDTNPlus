# Instalación
Para instalar la plataforma en su ultima versión hay que realizar los siguientes pasos:
```
git clone https://github.com/SeNDA-UAB/aDTN-platform.git
cd aDTN-platform
git checkout next
cmake .
sudo make install
ldconfig /usr/local/lib
```
